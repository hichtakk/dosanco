**Dosanco**: Simple DataCenter infrastructure database.

<img src="./docs/image/logo.png" width="300">

## Description
Dosanco manages IT infrastructure configurations bellow.

### Network
- Subnets
- IP address allocations
- VLANs

### Host
- Server/Node
- Group of hosts

### DataCenter
- Data Center
 - Floor
 - Data Hall
- UPS
- Row PDUs
- Rack PDUs
