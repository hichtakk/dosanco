Dosanco
=======

**Dosanco** is a network and server database, written in Go. :horse:

## Description
Dosanco manages IT infrastructure configurations bellow.

### Network
- Subnets
- IP address allocations
- VLANs

### Host
- Server/Node information

### DataCenter
- DCs
- Electrical facilities
- Rack PDUs
