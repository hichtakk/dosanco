Dosanco
=======

**Dosanco**: Simple DataCenter infrastructure database.

<img src="./docs/image/logo.png" width="350">

## Description
Dosanco manages IT infrastructure configurations bellow.

### Network
- Subnets
- IP address allocations
- VLANs

### Host
- Server/Node
- Group of hosts

### DataCenter
- DCs
- UPS
- Row PDUs
- Rack PDUs

## Usage
Dosanco privides command-line client.
